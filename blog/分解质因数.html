暴力解法 ​

???+ note "Tips" 最原始解法，复杂度为O(n)

cpp
for(int i = 2;i*i < n;i++){
    if(!(n%i)){
        while(!(n%i)){
            printf("%d*",i);
            n /= i;
        }
    }
}
printf("%d",n);
