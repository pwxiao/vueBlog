使用Vercel部署Typecho博客系统 ​

首先登录你的 GitHub，然后 fork 下面的存储库：typecho_vercel

登录 Railway 官网并注册一个数据库账号：

  1. 点击 Start a New Project

  2. 选择 Provision MySQL

  3. 点击 Connect 后滑至底部，找到 Available Variables，这里是你的数据库信息，复制并保存相关信息。

打开 GitHub 编辑器 ​

编辑 config.inc.php 文件，将数据库参数替换为之前创建的 Railway 数据库信息。

部署 ​

  1. 进入 Vercel https://vercel.com/

  2. 创建账号后点击 New Project

  3. 导入 fork 的 GitHub 存储库，然后点击 Build

    如果初次部署失败，进入 Vercel 的设置，修改 Node.js 版本为 18.x，然后再次部署。

    此时需要执行 Typecho 安装程序，在域名后加上 /install.php 进入安装程序。
