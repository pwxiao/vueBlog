使用 Barrier 实现 Mac 和 Windows 之间的无缝键鼠 ​

最近国家补贴购买了一台MacMini,准备将mac作为主力机开发，但是16g的内存在运行一些较大任务时难免捉襟见肘 于是想到在mac上写代码 程序包跑在另一台内存32gb，显卡4060的Windows笔记本上

Barrier 是一个开源的软件，可以让你在多台计算机之间共享键盘和鼠标。本文将介绍如何使用 Barrier 工具，实现 Mac 作为主机，Windows 作为客户端，实现无缝键鼠切换。

前提条件 ​

  • 一台运行 macOS 的电脑（作为主机）
  • 一台运行 Windows 的电脑（作为客户端）
  • 两台电脑在同一个局域网内

安装 Barrier ​

在 Mac 上安装 Barrier ​

  1. 打开 Barrier 的 GitHub 页面。
  2. 下载适用于 macOS 的安装包。
  3. 打开下载的 .dmg 文件，并将 Barrier 拖动到应用程序文件夹中。
  4. 打开 Barrier 应用程序。

在 Windows 上安装 Barrier ​

  1. 打开 Barrier 的 GitHub 页面。
  2. 下载适用于 Windows 的安装包。
  3. 运行下载的安装程序，并按照提示完成安装。
  4. 打开 Barrier 应用程序。

配置 Barrier ​

在 Mac 上配置 Barrier（作为主机） ​

  1. 打开 Barrier 应用程序。
  2. 选择“Server (服务器)”模式。
  3. 点击“Configure Server (配置服务器)”按钮。
  4. 在弹出的窗口中，将客户端（Windows 电脑）的屏幕拖动到合适的位置。
  5. 点击“OK”保存配置。

在 Windows 上配置 Barrier（作为客户端） ​

  1. 打开 Barrier 应用程序。
  2. 选择“Client (客户端)”模式。
  3. 在“Server IP (服务器 IP)”字段中，输入 Mac 电脑的 IP 地址。
  4. 点击“Start (启动)”按钮。

启动 Barrier ​

  1. 确保两台电脑都已经正确配置并连接到同一个局域网。
  2. 在 Mac 上点击“Start (启动)”按钮，启动 Barrier 服务器。
  3. 在 Windows 上点击“Start (启动)”按钮，连接到 Barrier 服务器。

遇到的问题 ​

  • 无法连接：关闭ssl，并且先启动服务端再启动客户端
